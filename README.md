# [Joy's Portfolio](https://joy-yang.netlify.app/)

## Tech Stack

### Frontend
- **React** – UI library for building the interactive interface.
- **Vite** – Fast build tool and development server.
- **Tailwind CSS** – Utility-first CSS framework for styling.
- **Fetch API** – Native browser API for making HTTP requests to the backend.

### Backend
- **Node.js** – JavaScript runtime environment.
- **Express** – Lightweight web framework for building RESTful APIs.
- **Axios** – Promise-based HTTP client for calling external APIs (Pexels, OpenAI/Gemini).
- **dotenv** – Manages environment variables (API keys, ports).

### External APIs
- **Pexels API** – Provides random, high-quality images based on search queries.
- **OpenAI API (GPT-4o mini)** – Generates descriptive captions for images.
- *(Alternative)* **Google Gemini API** – Can be used instead of OpenAI for caption generation.

### Deployment & DevOps
- **Netlify** – Hosts the frontend React app with continuous deployment from GitHub.
- **Render** – Hosts the backend Node.js/Express server, also connected to GitHub.

### Development Tools
- **Git & GitHub** – Version control and repository hosting.
- **npm** – Package manager for installing dependencies.
